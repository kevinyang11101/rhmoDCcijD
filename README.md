# 天气可视化系统 python887

## 项目概述

天气可视化系统是基于 Python 语言开发的一个项目，旨在提供天气数据可视化的功能，使得用户可以通过友好的界面直观地了解和对比不同地区的天气状况。

## 技术栈

- 后端：Python
- 前端：HTML, CSS, JavaScript
- 数据库：MySQL
- 框架：Django

## 功能模块

- **实时天气查询**：根据用户输入查询实时天气。
- **历史天气数据**：展示历史天气数据，支持数据可视化。
- **天气预报**：提供未来几天的天气预报。
- **数据导出**：允许用户导出查询的天气数据。

## 系统角色

- **普通用户**：查询天气，查看历史数据和天气预报。
- **管理员**：管理用户，维护系统。

## 运行环境

- 操作系统：Linux 或 macOS（推荐），Windows
- Python 版本：3.6 或以上
- 数据库：MySQL 5.7 或以上

## 部署步骤

1. 安装 Python 环境及必要的库。
2. 安装并配置 MySQL 数据库。
3. 克隆项目代码到本地。
4. 执行数据库迁移命令。
5. 运行 Django 开发服务器。
